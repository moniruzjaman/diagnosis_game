# 🌾 ফসল ফতোয়া (Crop Call)

> **আপনার ফসল, আপনার সিদ্ধান্ত!**  
> An engaging agricultural decision-making simulation game for rice disease identification and management.

---

## 🎮 What is Crop Call?

**ফসল ফতোয়া (Crop Call)** is an interactive agricultural simulation game where players step into the shoes of a farmer facing real crop challenges. Make critical decisions, identify rice diseases, and learn sustainable farming practices through an immersive gaming experience.

### Why "ফসল ফতোয়া" (Crop Call)?
- **ফসল (Fosol)** = Crop
- **ফতোয়া (Fotowa)** = Decision/Verdict
- **Crop Call** = The English equivalent emphasizing decision-making

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌾 **Interactive Field Scouting** | Explore fields in "W" pattern to find symptoms |
| 🔬 **Microscope Analysis** | Analyze samples with realistic lab simulation |
| 📏 **Damage Assessment** | Measure crop damage and learn about ETL |
| 🌤️ **Weather Evaluation** | Understand environmental risk factors |
| 🛡️ **IPM Decision Making** | Learn Integrated Pest Management |
| 🏆 **Achievements System** | Unlock 7 unique achievements |
| 📚 **Encyclopedia** | Access agricultural knowledge base |
| 🎮 **3 Difficulty Levels** | Easy, Medium, and Hard modes |
| 🔊 **Sound Effects** | Immersive audio feedback |

## 🎯 Game Phases

The game follows **CABI methodology**:

1. **Observe** - মাঠ পর্যবেক্ষণ (Field Scouting)
2. **Diagnose** - সমস্যা চিহ্নিতকরণ (Problem Identification)
3. **Measure** - ক্ষতির মাত্রা (Damage Assessment)
4. **Think** - ঝুঁকি পর্যালোচনা (Risk Evaluation)
5. **Act** - সিদ্ধান্ত গ্রহণ (Decision Making)
